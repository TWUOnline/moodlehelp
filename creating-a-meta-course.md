<div class="clarify-article">
<h2 class="clarify-article-title" style="font-size:30px; font-weight:bold; color:#333; padding:10px 0;">Making Meta Courses</h2>

<div class="clarify-article-description">
<p>Meta-Courses, (formerly known as Parent Courses) are your method of amalgamating multiple courses into one location. This is especially helpful when you have multiple courses with the same course content.</p>
</div>

<div class="clarify-steps-container">
<div class="clarify-step-container" id="clarify-step-1">
<h3 class="clarify-step-title" style="font-size:30px; font-weight:bold; color:#333; padding:10px 0;">Here are the steps.</h3>

<div class="clarify-step-instructions">
<p>First select the course you would like to use as the meta-course. Next, under the Administration block you will see &#39;Users&#39;, under Users select &#39;Enrollment Methods&#39;</p>
</div>

<div class="clarify-step-image-wrapper">
<div class="clarify-step-image-container" style="style='margin:20px 0;"><img alt="" class="clarify-step-image" height="757" src="http://media.screensteps.me/e-support/lzq4ay/here-are-the-steps-.png?1503010204" style="solid #ccc; padding:5px;" width="353" /></div>
</div>
</div>

<div class="clarify-clear">&nbsp;</div>

<div class="clarify-step-container" id="clarify-step-2">
<h3 class="clarify-step-title" style="font-size:30px; font-weight:bold; color:#333; padding:10px 0;">Next,</h3>

<div class="clarify-step-instructions">
<p>On the Enrolment Methods page you can see the amount of users currently in the course.&nbsp;</p>

<p>First, &#39;Add method&#39;, choose &#39;Course meta link&#39;.</p>
</div>

<h3 class="clarify-step-title" style="font-size:30px; font-weight:bold; color:#333; padding:10px 0;">Linking courses to your meta course.</h3>

<div class="clarify-step-instructions">
<p>First you can link courses by entering the course code in the &#39;Link course&#39; section of the page. Once you have added all the needed course(s), add these courses to a new group.</p>
