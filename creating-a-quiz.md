<h2 class="clarify-article-title">Creating a Quiz</h2>

<div class="clarify-article-description">
<p>Moodle 3 allows quizzes to be created within a course, instead of imported from an external source. Quizzes are a useful tool that can be utilized for many different purposes, and in many different formats. To learn more about how to create a quiz on MyCourses, follow this instructions below.</p>
</div>

<div class="clarify-steps-container">
<div class="clarify-step-container" id="clarify-step-1">
<h3 class="clarify-step-title">&nbsp;</h3>

<h3 class="clarify-step-title">Part 1: Creating a Quiz</h3>

<div class="clarify-step-instructions">
<p>&quot;<strong>Turn editing on</strong>&quot; in the top right hand corner.</p>

<p>Scroll to the section in which you would like to create a quiz; click on the &quot;<strong>Add an activity or resource</strong>&quot; button, select &quot;<strong>Quiz</strong>&quot; from the pop-up window, then click &quot;<strong>Add</strong>&quot;.</p>

<p>Next, you will be automatically redirected to the quiz creation page.</p>

<p>Supply a name for your quiz, and enter a description if you wish</p>

<p>&nbsp;</p>
</div>

<div class="clarify-step-container" id="clarify-step-2">
<h3 class="clarify-step-title">&nbsp;</h3>

<h3 class="clarify-step-title">Other options to utilize.</h3>

<div class="clarify-step-instructions">
<p>There are several options that can be utilized in the creation of your quiz. For example, the &quot;<strong>Timing</strong>&quot; options allow you to set a start and end date for quiz availability, as well as to set a time limit. The &quot;<strong>Grade</strong>&quot; options allow you to set a limit on allowed attempts.</p>

<p>You can set the peremeters for the quiz using the options listed below the &quot;<strong>General</strong>&quot; option.</p>

<p><strong>If you are uncertain about a quiz option, click on the question mark symbol next to each setting to learn more.</strong></p>

<p>Once you have adjusted the settings as desired, click &quot;<strong>Save and display</strong>&quot;. &nbsp;</p>

<p>Remember that you can edit your quiz name and settings at any time by returning to the course home page, clicking on the &quot;<strong>Edit</strong>&quot; dropdown list next to your quiz, and selecting &quot;<strong>Edit settings</strong>&quot;.</p>
</div>

<div class="clarify-step-container" id="clarify-step-3">
<h3 class="clarify-step-title">&nbsp;</h3>

<h3 class="clarify-step-title">Part 2: Adding Questions to a Quiz</h3>

<div class="clarify-step-instructions">
<p>Click on &quot;<strong>Edit quiz</strong>&quot;. This will take you to your <strong>&quot;Quiz bank</strong>&quot;.</p>

<p>On the &quot;<strong>Editing Quiz</strong>&quot; page, click on the &quot;<strong>Add</strong>&quot; dropdown menu on the right hand side, and select &quot;<strong>+ a new question</strong>&quot;.</p>

<p>A list of question types will be provided in a pop-up window. Each type of question works in a specific way; click on each one to learn more about it. Feel free to explore which question option is right for you.</p>

<p><strong>Note:</strong> If you use options such as &quot;<strong>Short Answer</strong>,&quot; students have to type in the exact answer that you have created in order to be graded correctly. For this reason, short answer questions may work best for one or two word answers. Essay questions can be utilized, but must be marked manually.</p>
</div>

<h3 class="clarify-step-title">&nbsp;</h3>

<h3 class="clarify-step-title">*For creating a Multiple Choice Question*</h3>

<div class="clarify-step-image-wrapper">
<div class="clarify-step-image-container"><img alt="" class="clarify-step-image" height="782" src="http://media.screensteps.me/e-support/reqnfh/-for-creating-a-multiple-choice-question--.png?1495739316" width="931" /></div>

<div class="clarify-step-instructions">
<p>There are several &quot;Choice&quot; options that you can create for the question. Fill in the first possible answer in the &quot;Choice 1&quot; box. If it is the correct answer, select &quot;100%&quot; from the &quot;Grade&quot; dropdown menu. If it is not the correct answer, select &quot;None&quot; from the &quot;Grade&quot; dropdown menu. If you chose &quot;Multiple answer allowed&quot; from the dropdown menu, make sure you select the appropriate grade breakdown for how much the particular &quot;Choice&quot; is worth. The &quot;Feedback&quot; box allows you to provide feedback to a particular question if desired. For example; &quot;Incorrect. See &#39;Chapter 2&#39; for more information on this question.&quot; After you have added all of the appropriate settings for &quot;Choice 1,&quot; move on to &quot;Choice 2&quot; and follow the same instructions as for &quot;Choice 1.&quot; Continue until you have filled in all of the possible answers that you would like to provide for this particular question.</p>
</div>
</div>

<div class="clarify-step-container" id="clarify-step-6">
<h3 class="clarify-step-title">&nbsp;</h3>

<h3 class="clarify-step-title">Once you have completed entering the information for your question &quot;Save changes&quot;.</h3>

<div class="clarify-step-image-wrapper">
<div class="clarify-step-image-container"><img alt="" class="clarify-step-image" height="398" src="http://media.screensteps.me/e-support/reqnfh/once-you-have-completed-entering-the-information-for-your-question--save-changes--.png?1495739316" width="907" /></div>
</div>

<div class="clarify-step-instructions">
<p><strong>Once you are back on the Question Bank page, you can make a few alterations if needed. </strong></p>

<ol>
	<li><strong>You can edit the quiz heading here.</strong></li>
	<li><strong>You can rearrange the position the question appaear if you don&#39;t wish for them to be shuffled.</strong></li>
	<li><strong>By clicking on this you can place page breaks between desired questions. </strong></li>
	<li><strong>&quot;Shuffle&quot; can be turned on or off; this options makes the provided answer options appear in a different order for each student. </strong></li>
	<li><strong>You may add additional questions.</strong></li>
	<li><strong>You can Preview or Delete the question.</strong></li>
	<li><strong>Enter the maximum mark here. </strong></li>
</ol>

<p>*Note: You may add as many questions to a quiz as you would like. You are also able to preview your quiz to have a sneak peak of what your quiz will look like to students. Go to your Administration block and select &quot;<strong>Preview</strong>&quot;. If you need to edit further after previewing your quiz, open the quiz and select &quot;<strong>Edit Quiz</strong>&quot; in your Administration block.</p>
